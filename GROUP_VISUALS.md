# ✅ Group Visualization Added

## "Imaginary Line" for Groups
You asked for a way to clearly see when an item is a Group.
I have added a visual indicator:

- **Group Selection:** When you select a Group, it now has a **Blue Dashed Border** (the "imaginary line").
- **Single Object:** Standard solid border.

This makes it instantly clear: "Am I holding a single text box, or a grouped set of items?" 
- **Dashes = Group**
- **Solid = Single Item**

## Summary of All Fixes
1. **Layer List:** Right sidebar shows all layers when nothing is selected. Perfect for finding hidden items.
2. **Access Layers:** You can click through transparent groups to reach items behind them.
3. **Group Visuals:** Dashed borders tell you exactly what is a group.

## Try It
1. Reload.
2. Select a Group (like the border content).
3. See the **Dashed Blue Line**. ✅
4. Select a simple text.
5. See the **Solid Line**. ✅
